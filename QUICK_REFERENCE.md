```bash
npm run dev
```

## 🎯 新功能

### 1. 新建收藏夹
- 位置：顶部导航栏
- 按钮：📁 新建收藏夹
- 功能：创建文件夹来组织收藏

### 2. 粘贴链接
- 位置：顶部导航栏
- 按钮：📎 粘贴链接
- 支持平台：知乎、微信、小红书、抖音、B站

### 3. 刷新列表
- 位置：顶部导航栏
- 按钮：🔄 刷新列表
- 功能：重新加载收藏列表

## 📝 常见问题

### Q: 背景图片不显示？
A: 
1. 访问测试页面检查
2. 强制刷新 (Ctrl+Shift+R)
3. 检查文件是否存在

### Q: 收藏失败？
A:
1. 打开控制台查看错误
2. 确认后端服务运行中
3. 检查网络请求详情

### Q: 按钮在哪里？
A: 所有按钮都在顶部导航栏，从左到右：
- 📁 新建收藏夹
- 📎 粘贴链接
- 🔄 刷新列表

## 📊 API端点

- `POST /api/collections` - 创建收藏
- `GET /api/collections` - 获取收藏列表
- `POST /api/folders` - 创建文件夹
- `GET /api/folders/tree` - 获取文件夹树
- `GET /api/health` - 健康检查

## 💡 提示

1. **强制刷新**: Ctrl+Shift+R
2. **打开控制台**: F12
3. **快速跳转**: Ctrl+G (输入行号)
4. **搜索文件**: Ctrl+P
5. **搜索内容**: Ctrl+F
